## Question: Neural Network Architecture Drawing and Notation

Consider a neural network with:

```text
input dimension 5 -> hidden layer 4 -> hidden layer 3 -> output layer 2
```

1. Draw the complete network with input layer, two hidden layers, and output layer.
2. Label the width of each layer.
3. What is the depth of this network if we exclude the input layer?
4. How many weight matrices does this network have?
5. How many bias vectors does this network have?
6. Write the shapes of all weight matrices using the row-vector convention.
7. Write the shapes of all bias vectors using the row-vector convention.
8. Count the total number of trainable parameters.
9. In your drawing, mark which objects are activations and which objects are trainable parameters.
