## Question: Confusion Matrix Scenario

A binary classifier predicts whether an email is spam. Use the following notation: TP, FP, FN, and TN.

1. Define TP in this spam-classification context.
2. Define FP in this context. Define FN in this context.
3. Define TN in this context. Draw a 2-by-2 confusion matrix with predicted labels on one axis and true labels on the other axis.
4. Write formulas for accuracy, precision, recall, and F1. Explain which error type is more annoying for a user: FP or FN. Justify your answer.

## Question: Multi-Class Extension

For multiclass classification, such as recognizing digits from 0 to 9, there are more than two possible labels. A confusion matrix can still summarize predictions.

1. What is the shape of a confusion matrix for 10-class digit classification?
2. What does a diagonal entry represent? What does an off-diagonal entry represent?
3. How can a confusion matrix reveal that the model confuses two specific classes?
4. Draw a small example showing confusion between digit 4 and digit 9.
