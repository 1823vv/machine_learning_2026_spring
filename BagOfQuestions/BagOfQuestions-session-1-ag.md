## Question: Polynomial Regression as Feature Engineering

A dataset shows a curved relationship between one input $x$ and a target $y$, so a straight line underfits. Polynomial regression creates new input features from powers of $x$ but still learns linear parameters.

1. Write degree-2 and degree-3 polynomial regression models.
2. For degree $k$, write the general polynomial regression model and define the feature transformation $\phi(x)$.
3. Explain why polynomial regression becomes multiple linear regression after feature transformation.
4. Explain why the model can be nonlinear in $x$ but still linear in parameters.
5. Draw three curves on the same dataset: degree 1 underfitting, medium degree good fit, and very high degree overfitting.
