## Question: Regression Metrics — MSE, MAE, and $R^2$

1. Write the mean squared error formula.
2. Write the mean absolute error formula using double pipes for absolute value. Explain why MSE penalizes large errors more strongly than MAE.
3. Explain why MAE can be easier to interpret in the original target unit. Write the $R^2$ formula or explain its meaning in words.
4. Give one situation where $R^2$ can be misleading.

## Question: Which Regression Metric Would You Use?

A model predicts apartment prices. Most predictions are close, but a few luxury apartments have very large prediction errors.

1. Which metric, MSE or MAE, is more sensitive to the luxury-apartment errors?
2. Which metric might be more robust to those few large errors? If large errors on luxury apartments are very costly, which metric might you emphasize?
3. Draw a small residual plot with a few large outliers.
4. Explain why reporting more than one regression metric can be helpful.
