## Question: Analyzing a Bivariate Normal Distribution Code Snippet

We have the following Python script using NumPy and Matplotlib to simulate and visualize a dataset representing two continuous random variables, $x$ and $y$:

```python
import numpy as np
import matplotlib.pyplot as plt

mu = [0, 0]
sigma = [[1.0, 0.5], 
         [0.5, 1.0]]
n_samples = 1000

data = np.random.multivariate_normal(mu, sigma, n_samples)

plt.figure(figsize=(6, 6))
plt.scatter(data[:, 0], data[:, 1], alpha=0.5)
plt.axis('equal')
plt.grid(True)

```

1. Identify what the `sigma` matrix represents. Explain what the diagonal elements (`1.0`) and the off-diagonal elements (`0.5`) tell us about the variables $x$ and $y$.

2. Describe the geometric shape and orientation of the scatter plot generated by this exact code. How does the off-diagonal value of `0.5` influence the direction of the data points?

3. Suppose we modify the off-diagonal elements of `sigma`. Name the specific relationship between $x$ and $y$, and describe the visual appearance of the resulting scatter plot, for each of the following three scenarios:
    * The off-diagonal elements are changed to `0.0`.
    * The off-diagonal elements are changed to `-0.5`.
    * The off-diagonal elements are changed to `0.95`.
