## Question: Visualizing and Analyzing a Bivariate Normal Distribution

We have the following Python script using NumPy and Matplotlib to simulate and visualize a dataset representing two continuous random variables, $x$ and $y$:

```python
import numpy as np
import matplotlib.pyplot as plt

mu = [0, 0]
sigma = [[1.0, 0.6], 
         [0.6, 1.0]]
n_samples = 1000

data = np.random.multivariate_normal(mu, sigma, n_samples)

plt.figure(figsize=(6, 6))
plt.scatter(data[:, 0], data[:, 1], alpha=0.6)
plt.axis('equal')
plt.grid(True)
plt.show()

```
